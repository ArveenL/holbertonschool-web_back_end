isinstance is a built-in py command to check if that particular object,
is an instance of a specific class or sub-class :

Syntax:
    It takes 2 arguments: isinstance(object, classinfo)

    - object: the specific object/variable you want to check
    - classinfo: A class, a type, or a tuple containing multiple classes/types

    It returns true if object/variable is of the same class/type/tuple 
    false if otherwise
